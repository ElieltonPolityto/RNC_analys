# Template de Caso RNC

Copie esta pasta, renomeie para `ID01`, `ID02`, etc., e coloque os arquivos reais dentro da nova pasta.

Arquivos esperados:

```text
ID01/
  projeto.pdf
  rnc.pdf
  metadata.json
  observacoes.txt
```

O nome dos PDFs pode variar. O indexador procura arquivos que tenham nomes contendo `projeto`, `project`, `rnc`, `nao_conformidade`, `não_conformidade`, `relatorio` ou `relatório`.
