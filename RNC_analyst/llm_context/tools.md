# Politica de consulta a manuais

Este arquivo so deve ser usado quando o app incluir manuais da pasta `tools`.

- Os manuais complementam a analise; eles nao substituem a evidencia do projeto atual.
- Use apenas os manuais anexados ao prompt pelo app.
- Nao cite catalogo, norma ou manual que nao esteja presente no contexto recebido.
- Quando houver conflito entre manual e padrao interno declarado em `FalsosPositivos.md`, trate como ponto para conferencia humana, nao como RNC automatica.
- Se um manual nao tiver dados suficientes para confirmar o item, informe a limitacao.
